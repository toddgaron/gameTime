Diving into Board Games

The board game market is exploding. Monopoly, Scrabble, Snakes and Ladders, and the other games that might have been available at the toy stores of our youth have been augmented with games like Catan, Race for the Galaxy, and other less familiar options. Since the late 50s with the advent of games like Diplomacy, a seven person World War I simulation, and Risk, a classic ruiner of friendships, the board game market has been booming.

This however presents a number of problems.

  1. With so many options buying a board game is difficult. I was with some friends a few months ago trying to buy a game and since we didn't know what we wanted, we ended up with something none of us really liked. Knowing what to buy is hard, in a store those colorful boxes blend into each other, and on the internet it is hard to discern if a reviewer's taste is anything like your own and if it will even be helpful. That's not even including that some games are significantly more expensive than others with games like Space Hulk, a personal favorite of mine, reaching into the two hundred dollar range.
  2. Another problem is that maybe you know what sort of game you want to play or design (for instance a wild west themed city-building game with variable player powers), but you want to know if that game exists or what games are most similar to it, if it doesn't.

One way to gain some traction is to look at board game data. I scraped the website BoardGameGeek, a large community of board game enthusiasts using their XML api and grabbed 30 GB of user collections and 600 MB of data on the games themeselves. The user data includes what games a user owns, wants, or wants to trade as well as numerical ratings. The game data includes properties like game length, number of players, or language difficulty as well as information like the number of comments on a game's message board or the theme and mechanics of a game.

One other way of looking at the data is using t-SNE, a toop to visualize high-dimensional data. The plot to the left shows all the games with at least 500 user ratings. The x-axis is roughly "how much is this a grand-strategy or wargame", with games like Europa Universalis all the way to the right and Guesstures on the far left. The y-axis is roughly "how abstract" a game is with games like Torres at the bottom of the axis and Zombies!!! at the top. The color and size reflect user rating and the number of ratings respectively.

Applying the same technique to subsets of user data tend to separate off wargamers from the remainder of users. Wargamers are a particular subgroup that we'll come back to later.

The plot below shows the BoardGameGeek rating for games with at least 500 user reviews as a function of year released. We see from the upper histogram that more games are being released per year. From the scatter plot we see that the mean score is increasing as well. It's unclear if games are getting better or mean scores are creeping up.

Within the dataset not every game is unique. There are hundreds of variations of Monopoly like a Star Wars themed version or one with ersatz credit cards instead of paper money. The number of Monopoly versions follow a roughly exponential curve as well. All the variations of a game like Monopoly are included.

Nor is every game playable alone. Some items in the data are expansions to a base game like the Cities and Knights expansion for Catan or new card sets for games like Apples to Apples or Cards Against Humanity. Some of these base games can be played alone; however many cannot. For example, the expansions to the game Smash Up are all playable with two players and need another expansion, or the base game, for more players. Expansions that cannot be played by themselves are tagged and excluded from the dataset.

Games in the set are also tagged with their properties by BoardGameGeek users.

Game Themes/Categories

Games within the BoardGameGeek dataset are tagged by users with their theme or category. This chord plot shows all the themes and categories with at least one hundred tagged games in the BoardGameGeek dataset, the lines connect any two themes that cooccur in at least ten games. The grouping is done by K-means. Clicking on a label will darken the lines that emanate from that that label and clicking on the solid bezel of the plot will remove all but the lines connecting to that portion of the bezel.

Looking at the plot we see themes like "humor" do not occur with themes like "World War II" or "space exploration." A theme like "card game" occurs with nearly every other tag.

Wargames

Wargames are an interesting encompassing both series miniature games like Warhammer 40k and tactical simulations like Advanced Squad Leader. The distribution of games that are either tagged as wargames or have a war theme is peaked to the right of games tagged otherwise. We might conclude that wargames are just on average better than other sorts of games, but a more likely explanation is that the subset of users that like wargames really like wargames.

Each game has its own message board and typically more comments per copy users own correspond to a lower average score (think Monopoly or Catan). This trend is true for wargames as well however the slop is much shallower and might conclude that this is a different subset of users who enjoy talking about wargames and might be the source of the different distribution.
